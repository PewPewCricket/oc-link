# OC-link
an rc service for OpenOS to create persistant symlinks

# Usage
All paths are absolute paths including: link names, files to link to, and paths to links <br>
create a symlink: ``rc link create <link name> <file to link to>`` <br>
create a temporary symlink: ``rc link createTemp <link name> <file to link to>`` <br>
remove a symlink: ``rc link remove <path to link>``
